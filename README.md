# ROKO - BASILISK ARCHIVE

Terminal game. Interactive fiction. Python 3.11+.

Pick a language at start (Portuguese / English). Use arrow keys and Enter. Esc to go back.

## Run

    pip install -r requirements.txt
    python main.py

Linux: `python3 main.py`
